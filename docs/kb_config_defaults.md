## kb config defaults

Set default project and column

### Synopsis

Set default project and column

```
kb config defaults [flags] [%%proj] [%col]
```

### Examples

```

# Set the default project to "mine" and default columnt to "backlog":
kb config defaults %%mine %backlog
```

### Options

```
  -h, --help   help for defaults
```
